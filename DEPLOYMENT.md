# Deployment Guide for Smart Way Kenya

## Introduction
This document outlines the steps to manually deploy the Smart Way Kenya project.

## Prerequisites
- Node.js v14 or higher
- npm (Node package manager)
- A compatible database (if applicable)

## Setting Up the Project
1. Clone the repository:
   ```
   git clone https://github.com/michaellolivery94-web/smart-way-kenya.git
   cd smart-way-kenya
   ```
2. Install dependencies:
   ```
   npm install
   ```

## Deployment Steps
1. Build the project:
   ```
   npm run build
   ```
2. Start the application:
   ```
   npm start
   ```

## Additional Notes
- Make sure to configure your environment variables according to your setup.
- Refer to the README for any other specific configurations needed.
